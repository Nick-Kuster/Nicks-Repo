1. Edit the contents of 'GoToDefinition/sql_repos.txt' to add or remove any directories you need or do not need. Leave the trailing semicolons. Key names do not matter at the moment, but could in future implementations.
2. Copy folder 'GoToDefinition' to 'C:\Program Files (x86)\Notepad++\plugins\Config'
3. Copy 'GoToDefinition.dll' to 'C:\Program Files (x86)\Notepad++\plugins'\GoToDefinition
4. Restart Notepad++
5. Open any SQL file to get started. Highlight a procedure or function name. Press CTRL + F12. A list of matching file names will pop up. Click on the one you want, and it will open in NotePad++!
